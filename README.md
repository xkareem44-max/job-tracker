🚀 Job Tracker

A modern, intuitive job application tracker designed to help you stay organized, focused, and in control throughout your job search.

✨ Overview

Job Tracker gives you a clear view of your entire job hunt — from applied positions to interviews and offers — all in one place. Built with a sleek UI and powerful tools, it helps you track progress, analyze performance, and move faster.

🔥 Features
📊 Dashboard
Get a quick overview of your job search with stats and visual insights.
📁 Application Management
Keep track of every job application, including status, company, and notes.
🧩 Kanban Board
Drag-and-drop interface to visualize your pipeline (Applied → Interview → Offer, etc).
👤 Profile Management
Store and manage your personal details and preferences.
🛠 Tech Stack
Category	Technology
Frontend	Next.js 14 (App Router)
UI	shadcn/ui + Radix UI
Styling	Tailwind CSS + Framer Motion
Backend	Supabase (Auth + Database)
Charts	Recharts
DnD	@dnd-kit
Forms	React Hook Form + Zod
Language	TypeScript
⚙️ Getting Started
1. Clone the repository
git clone <your-repo-url>
cd job-tracker
2. Install dependencies
npm install
3. Set up environment variables

Create a .env.local file in the root directory:

NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
4. Run the development server
npm run dev

Then open:
👉 http://localhost:3000
