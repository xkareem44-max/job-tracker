🧠 Overview

Job Tracker helps you manage your job search from start to finish.

Track applications, monitor progress, and visualize your pipeline — all in one clean interface.

✨ Features
📊 Dashboard
Visual stats and charts
Quick insight into your progress
📁 Application Management
Add, edit, and track job applications
Store company info, status, and notes
🧩 Kanban Board
Drag-and-drop workflow
Track stages (Applied → Interview → Offer)
👤 Profile
Manage personal info
Customize preferences
🛠 Tech Stack
Frontend → Next.js 14 (App Router)
UI → shadcn/ui + Radix UI
Styling → Tailwind CSS + Framer Motion
Backend → Supabase (Auth + Database)
Charts → Recharts
Drag & Drop → @dnd-kit
Forms → React Hook Form + Zod
Language → TypeScript
⚙️ Getting Started
1️⃣ Clone the repository
git clone <your-repo-url>
cd job-tracker
2️⃣ Install dependencies
npm install
3️⃣ Environment Variables

Create a .env.local file:

NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
4️⃣ Run the app
npm run dev

Open in your browser:
👉 http://localhost:3000
